# CharityOA
### 如何部署:
* 克隆仓库<br>
  `git clone https://github.com/dalewzm/CharityOA.git`
* 确认安装了virtualenv，pip, flask 以及处在正确地虚拟环境中
* 安装flask 拓展<br>
  `pip install -r requirements.txt`
* 本地运行<br>
  `python main.py runserver --host 127.0.0.1 --d`

### 代码结构
|config.py
|forms  表单类集合
|main.py
